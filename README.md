The built contract artifacts can be browsed via [unpkg.com](https://unpkg.com/browse/@draswap/v2-core@latest/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
